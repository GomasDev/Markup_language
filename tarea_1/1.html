Para usar estos apuntes debe ver tanto su visualización en el navegador, como su código fuente desde un editor o desde el propio navegador (bontón derecho de ratón-> menu contextual-> ver código fuente)

La estructura de HTML como lenguaje de marcas está basada en un conjunto de etiquetas anidadas, que pueden contener atributos. De esta forma la estructura que tendrán las etiquetas de forma genérica en su sintaxis será la siguiente:

<nombredelaetiqueta nombredelatributo1="valor del atributo1"...nombrelatributonN="valordelatributoN">Información contenida en la etiqueta</nombredelaetiqueta>

HTML es para la representación de información y estructura en la web. Pero necesita otros lenguajes para lo siguiente:

  • Visualización o estilo: se hace desde CSS
  • Interacción con el usuario: con el modelo de atributos de evento, junto con lenguaje de programación Javascript.

Ejemplos del modelo de eventos: cuando hacemos click en el botón sale una ventana alert con mensaje.
